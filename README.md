# MP3 Player PWA

A simple Progressive Web App (PWA) for playing MP3 files, deployable on GitHub Pages.

## Features
- Upload multiple MP3 files
- Play, pause, skip tracks, and navigate a playlist
- Progress bar with time display
- Offline support via service worker
- Installable as a PWA

## Setup
1. Clone the repository.
2. Ensure you have two icon files (`icon-192.png` and `icon-512.png`) in the `icons/` folder.
3. Deploy to GitHub Pages or serve locally with a tool like `http-server`.

## Deployment
- Push the project to a GitHub repository.
- Enable GitHub Pages in the repository settings, selecting the `main` branch and `/ (root)` as the source.
